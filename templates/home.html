You probably shouldn't use TweetNet

TweetNet is a tool to create auto-retweeting networks of Twitter accounts

When you create a TweetNet, you specify a user and a hashtag to watch. When that user tweets using that hashtag, any accounts in the TweetNet will automatically retweet the tweet.
